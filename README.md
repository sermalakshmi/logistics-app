# 🚚 Serma Logistics App

A full-stack logistics booking system built using Spring Boot and PostgreSQL.
Users can book transport services and admins can manage bookings through a dashboard.

---

## 🌍 Live Demo

👉 https://logistics-app-pfes.onrender.com

---

## ✨ Features

* 📦 Booking system for logistics services
* 📊 Admin dashboard to manage bookings
* 🔄 Status tracking (New, Confirmed, In Transit, Delivered)
* 📞 Contact integration
* 🎨 Modern responsive UI

---

## 🛠️ Tech Stack

* **Backend:** Spring Boot (Java)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** PostgreSQL
* **Deployment:** Render
